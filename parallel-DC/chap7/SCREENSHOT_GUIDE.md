## How to Take Screenshots

### For Windows:

#### Method 1: Using Snipping Tool
1. Open the application in your browser (http://localhost:5000)
2. Press `Windows + Shift + S`
3. Select the area you want to capture
4. Save as `screenshot.png` in the chap7 folder

#### Method 2: Using Print Screen
1. Open the application in your browser
2. Press `Alt + Print Screen` to capture the active window
3. Open Paint
4. Press `Ctrl + V` to paste
5. Save as `screenshot.png` in the chap7 folder

### For Terminal Screenshot:
1. Run the application with `python app.py`
2. Capture the terminal window showing the startup messages
3. Save as `terminal_screenshot.png` in the chap7 folder

## What to Capture

### Screenshot 1: Application Home Page (screenshot.png)
- Show the full task manager interface
- Include some sample tasks (both completed and pending)
- Make sure statistics are visible at the top
- Show the "Add Task" form

### Screenshot 2: Terminal Output (terminal_screenshot.png)
- Capture the terminal showing:
  - The command `python app.py`
  - The startup messages
  - The Flask development server running
  - The URL http://localhost:5000
